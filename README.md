# Вращающийся 3D-шар в Java (Swing)

Простой, но впечатляющий проект на чистом Java — вращающаяся 3D-сфера, отрисованная попиксельно с перспективой и простым освещением (Lambert shading).  
Всё работает без внешних библиотек, только стандартный JDK.
